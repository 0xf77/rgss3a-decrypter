# rgss3a Decrypter
Decrypts .rgss3a archive file created by RPG Maker VX Ace. Written in Python
This is an exercize on python bytes manipulation/operations. **Still a not working prototype**.


## Working features
- Gets files sizes, offesets and keys
- Decrypt file names
- "Extracting" (see todo)
